# The Bottom Feeder - Platform Specifications

> Technical requirements, character limits, and best practices for each platform.

## Instagram

### Content Specifications

| Type | Dimensions | Duration | File Size |
|------|-----------|----------|-----------|
| Feed Post | 1080x1350 (4:5) | - | 30MB max |
| Square Post | 1080x1080 (1:1) | - | 30MB max |
| Story | 1080x1920 (9:16) | 15 sec max | 30MB max |
| Reel | 1080x1920 (9:16) | 90 sec max | 4GB max |
| Carousel | 1080x1350 (4:5) | Up to 10 slides | 30MB each |

### Character Limits
- **Caption:** 2,200 characters (first 125 visible)
- **Bio:** 150 characters
- **Hashtags:** 30 max (10-15 recommended)
- **Username mention:** Counts toward limit

### Hashtag Strategy
**Primary (use always):**
- #TheBottomFeeder
- #CordlessPoolVacuum
- #PoolCleaning

**Secondary (rotate):**
- #PoolMaintenance
- #PoolLife
- #PoolOwner
- #BackyardGoals
- #PoolSeason
- #CleanPool
- #SwimmingPool

**Content-Specific:**
- #SatisfyingCleaning (for clean videos)
- #BeforeAndAfter (for transformations)
- #PoolTips (for educational)
- #PoolPro (for professional content)

### Best Practices
- First line hooks: Question or bold statement
- Line breaks for readability
- CTA at the end
- Hashtags: In caption or first comment
- Tag location when relevant
- Use alt text for accessibility

## YouTube

### Content Specifications

| Type | Dimensions | Duration | File Size |
|------|-----------|----------|-----------|
| Standard Video | 1920x1080 (16:9) | No max | 256GB max |
| Shorts | 1080x1920 (9:16) | 60 sec max | - |
| Thumbnail | 1280x720 | - | 2MB max |

### Character Limits
- **Title:** 100 characters (70 recommended)
- **Description:** 5,000 characters
- **Tags:** 500 characters total

### SEO Best Practices
- Front-load keywords in title
- Detailed description with timestamps
- Custom thumbnails (bright, faces, text)
- End screens for related content
- Cards for products/links
- Closed captions for accessibility

### Content Structure
```
0:00 - Hook (5-15 sec)
0:15 - Intro (what you'll learn)
0:30 - Main content
[X]:00 - Call to action
[X]:15 - End screen
```

## Email Marketing

### Technical Specs
- **Subject line:** 40-60 characters
- **Preview text:** 35-90 characters
- **Body width:** 600px max
- **Image width:** 600px max
- **File size:** Under 100KB per image

### Best Practices
- Mobile-first design (60%+ opens on mobile)
- Single column layout
- Large tap targets (44x44px min)
- Alt text for images
- Plain text version
- Clear unsubscribe

### Subject Line Formulas
- Question: "Is your pool ready for summer?"
- How-to: "How to clean your pool in half the time"
- Number: "5 mistakes pool owners make"
- Curiosity: "The secret pool pros won't tell you"
- Personal: "[Name], your pool deserves better"

## Blog/Website

### Content Specifications
- **Featured image:** 1200x630
- **In-content images:** 800px wide
- **Infographics:** 800px wide, unlimited height
- **Video embeds:** 16:9 aspect ratio

### SEO Requirements
- **Title tag:** 50-60 characters
- **Meta description:** 150-160 characters
- **URL slug:** Short, keyword-rich
- **H1:** One per page, keyword included
- **H2/H3:** Logical hierarchy
- **Alt text:** Descriptive for all images

### Content Structure
```markdown
# H1: Main Title (with keyword)

Introduction paragraph (hook + what they'll learn)

## H2: First Main Section
Content...

### H3: Subsection if needed
Content...

## H2: Second Main Section
Content...

## H2: Conclusion / Next Steps
CTA to product or related content
```

## Video Production Guidelines

### General Specs
- **Resolution:** 1080p minimum, 4K preferred
- **Frame rate:** 30fps (60fps for action)
- **Audio:** Clear, minimal background noise
- **Lighting:** Well-lit, natural preferred

### Branding Elements
- Logo intro: 2-3 seconds max
- Lower thirds: Brand colors (#0077B6, #00B4D8)
- End card: CTA + logo
- Watermark: Optional, subtle

### Thumbnail Guidelines
- Bright colors (contrast with YT/TikTok UI)
- Face showing emotion when possible
- Large readable text (3-5 words)
- Before/after split when relevant
- Consistent style across series
