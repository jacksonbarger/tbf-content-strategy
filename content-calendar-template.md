## Hootsuite Scheduling Notes

### Batch Creation Workflow
1. **Monday AM:** Create week's content
2. **Monday PM:** Schedule all posts in Hootsuite
3. **Daily:** Monitor and engage
4. **Friday:** Review week's performance
5. **Friday:** Plan next week's themes

### Hootsuite Labels
Use consistent labels for organization:
- `educational` - Tips and how-tos
- `social-proof` - Testimonials and reviews
- `product` - Product features and demos
- `community` - UGC and engagement
- `promotional` - Sales and offers
- `seasonal` - Holiday and seasonal content

### Bulk Upload Format
For CSV uploads:
```
Date,Time,Message,Image URL,Platform
2024-01-15,11:00,Post caption here,https://image.url,instagram
```

## Content Recycling Schedule

Evergreen content can be recycled every 90 days:

| Content Type | Recycle After | Update Required |
|--------------|---------------|-----------------|
| How-to guides | 90 days | Check for accuracy |
| Product features | 60 days | Fresh visuals |
| Testimonials | 90 days | None |
| Tips & tricks | 60 days | New examples |
| Before/after | Never recycle | Always fresh |

---

## Approval Workflow

### Standard Posts
1. Create → 2. Self-review → 3. Schedule

### Promotional Posts
1. Create → 2. Team review → 3. Approve → 4. Schedule

### Crisis/Sensitive
1. Draft → 2. Leadership review → 3. Legal (if needed) → 4. Post
